# Components

Componentes reutilizables organizados por módulo funcional.

## Estructura

- `ui/` - Componentes base de interfaz (botones, inputs, cards, etc.)
- `3d/` - Componentes relacionados con Three.js y escenas 3D
- `game/` - Componentes específicos del motor de juego
- `auth/` - Componentes de autenticación (login, registro, etc.)
- `dashboard/` - Componentes del panel de control para docentes
